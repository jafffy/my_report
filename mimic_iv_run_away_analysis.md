## 3. 핵심 발견: 왜 도망갔는가?

### 3-1. Z5321 환자 분석 (127건)

Z5321 = "Procedure not carried out due to patient leaving prior to being seen"

| 입원 유형 | 건수 | 비율 |
|-----------|------|------|
| Observation Admit | 53건 | 41.7% |
| EW EMER. | 30건 | 23.6% |
| EU Observation | 29건 | 22.8% |

→ 대기 중 떠난 환자들로, **진정한 의미의 도망** 케이스

## 4. K-MIMIC vs MIMIC-IV 비교

| 항목 | K-MIMIC | MIMIC-IV |
|------|---------|----------|
| **총 Run away 케이스** | **409건** | **322건** |
| 응급실 비율 | 86% | 84.4% (ED+Obs) |
| 주요 정신과 진단 | 우울증, 조현병, 알코올의존 | 양극성, 불안, PTSD |
| 병원 집중도 | 병원 201 (87%) | 분산됨 |

### 공통점
- 응급실/관찰 입원 환자가 대다수
- 정신과 진단 환자 비율 높음
- 알코올/약물 관련 문제

### 차이점
| K-MIMIC | MIMIC-IV |
|---------|----------|
| 조현병(F20) 비중 높음 | 양극성 장애(F31) 비중 높음 |
| 단일 병원 집중 | 여러 병원 분산 |
| 알코올 위주 | 오피오이드/코카인 포함 |

## 5. 결론 및 해석

### Run away의 주요 원인 추정:

1. **정신건강 문제 (가장 주요)**
   - 양극성 장애 환자의 조증 삽화 시 충동적 이탈
   - 경계성 인격장애 환자의 충동성
   - 조현병 환자의 병식 부족
   - PTSD, 불안장애 환자의 병원 환경 회피

2. **알코올/약물 관련 문제**
   - 알코올 의존/금단 환자의 음주 욕구
   - 오피오이드/코카인 의존 환자의 약물 갈망

3. **사회경제적 요인**
   - Medicaid 51.6% (저소득층)
   - 대기 시간 불만
   - 치료비 우려

4. **응급실/관찰병동 특성**
   - 84.4%가 응급실 또는 관찰 입원
   - 비자발적 내원 후 이탈
   - 대기 중 떠남 (Z5321)
